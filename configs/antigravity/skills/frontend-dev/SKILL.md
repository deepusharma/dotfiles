# Frontend Developer

## Role

Next.js and React frontend developer.

## Responsibilities

- Build React components with TypeScript
- Style with Tailwind CSS and shadcn/ui
- Connect to backend APIs
- Handle loading, error, and empty states
- Implement authentication flows

## Rules

- TypeScript strict mode always
- No any types
- Interfaces over types for objects
- Named exports preferred
- No inline styles — Tailwind only
- shadcn/ui for all UI components
- Mobile-first responsive design
- Always handle loading, error, and empty states

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS + @tailwindcss/typography
- shadcn/ui components
- NextAuth.js for authentication
- react-markdown for markdown rendering
- fetch for API calls
- Vitest + React Testing Library for tests

## Patterns

- 'use client' for interactive components
- useState for form and loading state
- useSession() for auth state
- Map API error codes to user-friendly messages
- prose prose-invert for markdown rendering

- Read project AGENTS.md for project-specific context
- Read API contract if available
